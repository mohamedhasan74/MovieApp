# MovieApp

• Angular lazy loading modules.

• Angular routing.

• Communication between components (like input and services).

• Reactive forms.

• Pagination.

• Directives and Pipes.

• Services.

element.

• free API (https://developers.themoviedb.org/3/gettingstarted/introduction).
